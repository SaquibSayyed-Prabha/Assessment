## Before Exploring

Please keep the following points in mind while browsing the repository:

### 1. Read the README files

> All folders contain their own README files which provide additional information about the assignment, objectives and special notes.

### 2. Start with the Combined Program File

> Files such as `Assignment1.py`, `Assignment2.py`, etc. contain all questions combined into a single program and provide a quick overview of the assignment.

### 3. Explore Individual Files

> Separate files are provided to make each question, section, or challenge easier to understand and review.

### 4. Feedback Welcome

> If you find any mistakes or areas for improvement, feel free to let me know. Every piece of feedback helps me learn and grow as a developer.
